# URL Shortener

Full-stack application for shortening long URLs.

## Структура проекта

- `back-end/` - Spring Boot application (API)
- `front-end/` - React application (UI)

## Запуск проекта

### Back-end
```bash
cd back-end
./gradlew bootRun
```

### Front-end
```bash
cd front-end
npm start
```
